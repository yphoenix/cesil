# cesil
CESIL Language Interpreter in JavaScript

## Install
### Globally
```
npm install -g package.json
```

### Run (globally)
```
cesil file.ces
```

### Run (locally)
```
npm run cesil file.ces

// or

node src/cesilCLI.js file.ces

// or

```
