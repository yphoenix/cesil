# cesil
CESIL Language Interpreter in JavaScript

## Install
### Globally
```
npm install -g cesil
```
### Run (globally)
```
cesil file.ces
```

### Locally
```
npm install cesil
```
### Run (locally)
```
npm run cesil file.ces

// or

node src/cesilCLI.js file.ces
```
